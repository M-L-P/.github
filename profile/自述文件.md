# M.L.P.
M.L.P. 业余时间致力于开机与引导的图形界面。<br/>
让使用电脑的过程"凤头豹尾"。<br/>
|___凤头豹尾___|
|--|
|![image](https://github.com/M-L-P/.github/assets/69227436/cc4bca32-97e5-4a4c-8d08-991112749a1d)|

开端如凤首美丽，结尾如豹尾有力；<br/>
身躯如猪，肉有肥瘦。<br/>

上网时，能检索需要的信息，却也穿插广告；<br/>
关机，屏幕快速变黑熄屏，为你节省时间。<br/>
